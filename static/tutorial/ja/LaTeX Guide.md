# LaTeX のインストール

Zettlr は LaTeX と最もよく連携します。LaTeX は無料で使用でき、すべてのオペレーティングシステムで動作します。すべてのファイルをエクスポートし、同僚とよりスムーズに作業できるようにするために、できるだけ早くインストールすることをお勧めします。Markdown に移行した後でも、Word 文書を扱ったり、Markdown を使用しない人々と意見を共有したりする必要が出てくるでしょう。

LaTeX は、すべてのプラットフォームで標準のインストーラを使用してインストールできます。推奨されるディストリビューションは以下の通りです：

* Windows: [MikTeX](https://miktex.org/download)
* macOS: [MacTex](https://www.tug.org/mactex/morepackages.html) (_フルバージョンよりもはるかにサイズが小さい Basic Tex をインストールするだけで十分です!_)
* Linux: [TeX Live](https://www.tug.org/texlive/) (`texlive-base` パッケージをインストールしてください。`texlive-xetex` パッケージも必要になる場合があります。)
