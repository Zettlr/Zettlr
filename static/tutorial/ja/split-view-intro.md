# 分割表示

Zettlr は分割表示に対応しており、作業効率を高めることができます。この機能を示すために、3 つのエディタペイン（左側に 1 つ、右側に上下に積まれた 2 つ）を使用したサンプルレイアウトを用意しました。ファイルをドラッグ＆ドロップして、新しい分割を作成できます。仕組みは以下の通りです：

* 新しいファイルは、常に最後にアクティブだったペイン（つまり、カーソルがあるペイン）で開きます
* ドキュメントタブをタブバーの外にドラッグすると、ファイルを移動できます
* タブをタブバー上またはエディタの中央部分に移動すると、ファイルは対応するペインに移動します
* タブをいずれかのエディタペインの端（上、右、下、左）に移動すると、指定されたペインがその方向に分割されます
* ペインの最後のファイルを閉じるか移動すると、そのペインは自動的に閉じます
* Zettlr は複数ウィンドウにも対応しています。`Cmd / Ctrl+Shift+N` を押すと、新しいウィンドウが開きます
