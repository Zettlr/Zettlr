# Zettlr へようこそ！

 ![zettlr.png](./zettlr.png)

こんにちは、Zettlr をお選びいただきありがとうございます！ 🎉 このインタラクティブなチュートリアルは、すべてのドキュメントに目を通さなくても、すぐにアプリを使い始められるようにまとめられています。しかし、Zettlr はこの短い入門書で概説されている以上の多くの機能を備えているため、[🔗 詳細なドキュメント](https://docs.zettlr.com/)(リンクを開くには `Cmd` または `Ctrl` を押しながらクリックしてください) もぜひご覧になることをお勧めします。ドキュメントは多くの言語で利用可能で、豊富な機能セットをナビゲートするのに役立ちます。オンラインドキュメントは、`F1` キーを押すか、ヘルプメニューの該当項目を選択することでいつでも開くことができます。

さて、それでは始めましょう！

> このチュートリアルでは、クリックして移動できるウェブリンクが表示されます。デフォルトでは、リンクをクリックすることは「このリンクを編集したい」という意味に解釈されます。リンクを開きたい場合は、macOS を使用している場合は `Cmd` キーを、それ以外の場合は `Ctrl` キーを押しながらリンクをクリックしてください！ `Cmd/Ctrl` と表示されている場合は、「macOS の場合は `Cmd` を、それ以外の場合は `Ctrl` を使用する」という意味としてお読みください。

## このチュートリアルについて 🎬

多くの Markdown アプリケーションでは、このようなインタラクティブなチュートリアルを活用して、アプリケーションの使用方法を実践的に学ぶことができます。Zettlr では、ドキュメントフォルダー内に "Zettlr Tutorial" という小さなディレクトリを作成し、それを読み込んだ状態で提供しています。今、左側のサイドバー (「ファイルマネージャー」と呼ばれます) にそのディレクトリの内容が表示されています。見てみましょう！

ご覧の通り、"Workspaces" という見出しがあり、その下にチュートリアルフォルダー – Zettlr Tutorial – があり、現在のファイルがハイライト表示されています。また、他のファイル、例えば "references.json" も表示されています。これはもうすぐ重要になります。このファイルマネージャーのデフォルトのモードは「combined」と呼ばれ、お使いのシステムのファイルブラウザと同様に、ファイルとフォルダーが混在して表示されます。Zettlr のファイルマネージャーには、ファイルとフォルダーを分けて表示する「thin」と expanded の 2 つの他のモードがあり、設定で有効にすることができます。

"Zettlr Tutorial" フォルダーは「ワークスペース」と呼ばれ、現在アプリに読み込まれているすべてのファイルが含まれています。追加のワークスペースや、スタンドアロンのファイル (どのワークスペースにも属さないファイル）) を開くこともでき、それらもファイルマネージャーに表示されます。

ディレクトリを右クリックすると、「閉じる」か「削除」することができます。Zettlr でワークスペースやトップレベル (スタンドアロン) のファイルを_閉じる_ということは、アプリケーションからアンロードするという意味で、ファイルはコンピュータ上に残ります。_削除_すると、Zettlr はそのファイルをごみ箱に移動します。しかし、まだ学ぶべき新しいことがあるので、今は実行しないでください！ ✍🏼

> 注：一部の Linux ディストリビューションでは、Zettlr がファイルをごみ箱に移動するために特定のパッケージに依存しているため、ごみ箱への移動がすぐに機能しない場合があります。ファイルやフォルダーの削除に問題が発生した場合は、[FAQ セクション](https://docs.zettlr.com/en/faq/#im-using-linux-and-deleting-files-doesnt-move-them-to-the-trash)を確認してください。そこに解決策が記載されています。

Zettlr はワークスペースの概念を中心に構築されています。そのため、常に少なくとも 1 つのワークスペースディレクトリを開いて、その中で作業を行うことをお勧めします。ワークスペースはアプリを開くたびに自動的に読み込まれ、今表示されているディレクトリツリーに常に表示されます。

実際のノートを保存したいフォルダがすでにある場合、または既にいくつかのノートがある場合は、ツールバーの「ワークスペースを開く...」ボタンまたはショートカット `Cmd/Ctrl`+`O` を押して、今すぐ開くことができます。

## Markdown の使い方 📝

Zettlr は Markdown エディタです。つまり、Microsoft Word、LibreOffice、Apple Pages など、おなじみのアプリとほぼ同じように動作します。しかし、無数のツールバーボタンをクリックする代わりに、文字だけを使って要素に構造を適用することができます。つまり、キーボードから手を離す必要がありません！ ✨

最も重要な要素を簡単に見てみましょう：

1. テキストを **bold** や _italic_ にするには、アンダースコアまたはアスタリスクで囲みます。どちらを選ぶかは完全に自由です。1 文字でイタリック、2 文字でボールド、3 文字で __*ボールドとイタリックの両方*__ になります。
2. 見出しはハッシュタグのように作成します – 単に `#` 文字の後にスペースを入力します。レベル 1 から 6 までの見出しを作成するには、最大 6 つの `######` 文字を使用できます。
3. リストは文字通り作成されます – 新しい行に `*`、`-`、または `+` を書くだけです。番号付きリストは、数字の後にドットを付けます。
4. 最後に、引用符はメールで引用テキストが表示されるのとまったく同じように記述します。単に `>` を使って区切ります。

もちろん、他にもたくさんの要素があります。例えば脚注 – これにカーソルを合わせてみてください。[^1] このチュートリアルでは、Zettlr が真の学術作業や、Zettelkasten を使ったナレッジマネジメントを可能にするために使用する特別な要素についても学びます。

## リンク ⛓

学術テキストではあまり頻繁には使用されませんが、リンクは Markdown の強力なツールであり、Zettlr はそれをさらに進化させています。Zettlr はリンクに関して非常に賢く動作します。さっそく作ってみましょう！次のリンクをクリップボードにコピーします：https://fosstodon.org/@zettlr

次に、次の単語を選択します：「link to our Mastodon account」そして `Cmd/Ctrl+K` を押します！ Zettlr はクリップボードに有効なウェブリンクがあることを認識し、それを自動的にリンク先として使用します。さらに、Zettlr はテキストを読みやすくするために、リンク先を自動的に非表示にし、リンクテキストのみを表示します。Zettlr がデフォルトでレンダリングする多くの要素の中に気に入らないものがあれば、「表示」環境設定で 1 つずつオフにすることができます。

しかし、Zettlr は一般的なウェブリンクだけでなく、コンピュータ上のどこかにあるファイルにリンクすると、そのようなリンクをクリックした場合にファイルを開くこともできます。一般的に、Zettlr はリンクに関してだけでなく、執筆体験を可能な限りスムーズにすることを目指していることを覚えておいてください。

## サイドバー 📎

Markdown の基本を理解したところで、Zettlr ができるその他の機能をいくつか紹介します！ ツールバーの右上にあるカラムのようなアイコンをクリックしてください。サイドバーが開き、4 つのタブが表示されます。

最初のタブには、動的に生成される目次が表示されます。見出しをクリックすると、テキスト内のその場所に移動できます。2 番目のタブには、参考文献のリスト (現在のドキュメントに存在する場合) が表示されます。これは、ドキュメント内で何を引用したかを確認するための強力な方法として機能します。

> 参考文献は、組み込みの引用スタイルを使用してのみ書式設定されることに注意してください。ドキュメントをエクスポートする際に、「エクスポート」環境設定で引用スタイルを設定している場合 (詳細は後述)、Zettlr はその選択されたスタイルを使用します。

3 番目のタブには、関連ファイル、つまり Zettlr が現在のファイルに関連していると判断したファイルが表示されます。これは、すべてのファイル内で使用されているタグと内部リンク (詳細は後述) を調べることによって行われます。このリストの上部には、現在表示しているファイルにリンクしているファイルが表示されます。その後、現在のファイルと共通のタグを持つファイルが続きます。現在のファイルと共通するタグが多いほど、リストの上位に表示されます。

最後のタブには、現在選択されているディレクトリに存在するすべての非 Markdown ファイルが表示されます。"LaTeX Guide.pdf" というファイルが表示されています。これが何か知りたいですか？見てみましょう：それをクリックすると、デフォルトの PDF ビューアでファイルが開きます！

## インタラクティブな要素 ⏯

これまでに、Zettlr について多くのことを学びました。すべてのチェックボックスにチェックを入れることができますか？

- [ ] ファイルとディレクトリの操作
- [ ] Markdown の基本を学ぶ
- [ ] ファイルをエクスポートするための LaTeX をインストールする

エディタ内のいくつかの要素は、チェックボックスのようにインタラクティブです。表も高度なインタラクティブ性を持つ要素の一つです。次の表を見てください：マウスをホバーすると、表を操作できるいくつかのボタンが表示されます！

| File                        | Purpose                                                          | Filename        |
|-----------------------------|------------------------------------------------------------------|-----------------|
| Welcome to Zettlr!          | Gives a basic overview over the abilities of Zettlr              | welcome.md      |
| Working with a Zettelkasten | Introduces the various Zettelkasten-features of Zettlr           | zettelkasten.md |
| Citing with Zettlr          | Highlights the capabilities of working with references databases | citing.md       |

列の配置や、行や列の追加と削除ができます。表エディタは、常に現在アクティブな列または行に対して動作します。したがって、特定の行を削除するには、その行内のいずれかのセルが選択されていることを確認してください。動作に慣れるために、表で少し遊んでみてください！
